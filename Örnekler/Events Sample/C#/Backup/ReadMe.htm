Events Sample

This sample shows how to declare, invoke, and configure events in C#. For additional information, see Events (C# Programming Guide).

To build and run the Events samples within Visual Studio

  1. In Solution Explorer, right-click the Events1 project and click Set as StartUp Project.

  2. On the Debug menu, click Start Without Debugging.

  3. Repeat the preceding steps for Events2.

To build and run the Events samples from the Command Line

  1. Use the Change Directory command to change to the Events1 directory.

  2. Type the following: 

    csc events1.cs
events1

  3. Use the Change Directory command to change to the Events2 directory.

  4. Type the following: 

    csc events2.cs
events2
