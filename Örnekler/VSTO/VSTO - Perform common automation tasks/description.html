Introduction

As you develop an Excel add-in, you'll likely need your add-in to perform a myriad of automation tasks. For example, your add-in might copy, delete, move, or hide worksheets, groups rows of data, or format groups of cells. The possibilities are endless. This sample features 27 code snippets that help you get started with some of the most popular automation tasks.

These tasks include the following:

Workbook tasks

  • Save and close a workbook
  • Protect and unprotect a workbook
  • Set a password on a workbook
  • List recently used workbooks
  • Email a workbook

Worksheet tasks

  • Add a worksheet
  • Copy a worksheet
  • Move a worksheet
  • Print a worksheet
  • List worksheets in a workbook
  • Protect and unProtect a worksheet
  • Add, delete or hide a comment in a worksheet
  • Check spelling in a worksheet

Cell range tasks

  • Populate ranges of cells
  • Sort ranges of cells
  • Sort tables (list objects) columns
  • Store and retreive a date in a range
  • Apply styles to ranges of cells
  • Run calculations
  • Display the current user's log-in information in a cell.

If you're building an workbook customization for Excel instead of an add-in, you can find a version of these snippets that can be used in the code of your Excel workbook or workbook tamplate project here - VSTO – Perform common automation tasks in an Excel workbook customization.

.

Building the Sample

This sample contains only the automation snippets. It does not contain code that runs them. However, you can use this sample to give you ideas and help you get started using the Excel object model in an Excel add-in.
