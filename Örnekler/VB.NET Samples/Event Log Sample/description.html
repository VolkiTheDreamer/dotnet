Event Log Sample

Visual Basic offers two different logging mechanisms:

  • The component, used in this sample, provides access to the operating system event logs. You can add and delete logs, add and remove event log sources, write messages, and delete messages.

  • The My.Application.Log object also provides access to some of the features. In addition, My.Application.Log has features that enable you to write messages to text files. For more information, see Logging Sample and .

Demonstrates

The main form, Form1, contains three controls to read, write, and create or delete event logs. Each opens a new form that collects information specific to the operation.

The form WriteForm collects the text, ID, and type for an event-log entry and writes the entry to the Application event log using the method.

The form ReadForm fills a control with the names of the event logs on the computer. The list is obtained by calling the method, and then displaying the value of the property for each log. The last ten entries of the selected log are retrieved with property and then displayed in a control.
