Demonstrates

This application contains two classes, one with standard serialization and one with custom serialization. The user interface includes buttons to serialize and deserialize the two classes. The class with standard serialization can be serialized to either a SOAP file or a binary file. The class with custom serialization is serialized to a SOAP file. A control is used to display the SOAP files. The two classes both have the attribute. The classes are described in the following table.
