Demonstrates

This sample consists of a main form that is used to start four different applications:

<![if !supportLists]> ·         <![endif]>The Five Stones application is a software version of a classic board game. Line and shape controls are used to draw the user interface. Following are some of the techniques demonstrated:

<![if !supportLists]> ·         <![endif]>The SurfaceLoad procedure demonstrates how to create shape controls at run time.

<![if !supportLists]> ·         <![endif]>The StartNewGame procedure demonstrates how to make shape controls invisible but still enable them to respond to a Click event.

<![if !supportLists]> ·         <![endif]>The DrawButton procedure demonstrates how to use a RectangleShape control as a button.

<![if !supportLists]> ·         <![endif]>The FillColor, FillGradientColor, and FillGradientStyle properties are used in several locations at design time to demonstrate how to create a gradient effect.

<![if !supportLists]> ·         <![endif]>The Darts application is a simple dart game that uses Line and Shape controls. Following are some of the techniques demonstrated:

<![if !supportLists]> ·         <![endif]>The dart image is created by using the BackgroundImage property of a RectangleShape control. It uses the MakeTransparent method of the Bitmap object to mask the background of the dart image at run time.

<![if !supportLists]> ·         <![endif]>Two Timer controls are used to animate the dart at run time.

<![if !supportLists]> ·         <![endif]>The GetChildAtPoint method of the ShapeContainer class is used to determine which control is under the dart when it is fired.

<![if !supportLists]> ·         <![endif]>The Picture application demonstrates how to draw a complex picture by using a combination of Line and Shape controls and graphics methods. Following are some of the techniques demonstrated:

<![if !supportLists]> ·         <![endif]>The GraphicsPath object is used to create irregularly shaped Line and Shape controls at run time.

<![if !supportLists]> ·         <![endif]>The HatchBrush object is used to provide various textures and patterns on the Shape controls at run time.

<![if !supportLists]> ·         <![endif]>The FillColorFillGradientColor, and FillGradientStyle properties are used to demonstrate solid and gradient fills on Shape controls.

<![if !supportLists]> ·         <![endif]>Timer controls are used to animate the picture at run time.

<![if !supportLists]> ·         <![endif]>The Math Game application is an example of a simple quiz application. If the user interface looks a bit primitive, consider that it was designed by the eight-year-old daughter of one of our developers. Although it does not demonstrate any special techniques, it does demonstrate how easy it is to create an application that uses Line and Shape controls.
