PrintForm Sample

To get samples and instructions for installing them, see the following:

  • Click Samples on the Visual Studio Help menu.

    For more information, see Locating Sample Files.

  • The most recent versions and the complete list of samples are available on the Visual Studio 2008 Samples Web site.

  • You can also locate samples on your computer's hard disk. By default, samples and a Readme file are copied to a folder under \Program Files\Visual Studio 9\Samples\. For Visual Studio Express Editions, all samples are located on the Internet.

To run this sample

  • Press F5.

Demonstrates

This sample uses a PrintForm component to demonstrate how to print an image of a graphical form. The PrintAction property is used to display the print output in a PrintPreviewControl dialog box. You can change this property value to PrintToPrinter to send the output to a printer, or to PrintToFile to send the output to a PostScript file.

 

The sample also demonstrates the use of the graphics techniques that are used to create the text and graphics on the form. This includes code for drawing a pie chart.
