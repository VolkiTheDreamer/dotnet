Demonstrates

Windows Services are long-running programs that perform their tasks without user interaction. This sample uses the class from the namespace. This class enables you retrieve a list of the running services through a shared method, . By using the class, you can start, stop, pause, or resume a service.

Using the user interface presented you can start, stop, pause, and resume a service. The application checks the current state of the service to see what buttons should be enabled. The Relist All Services command on the Actions menu calls the method to fill the control with the processes. The Refresh command on the Actions menu updates the status of each process already listed in the control.
